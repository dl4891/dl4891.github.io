1. Introdução ao Sistema

O Sistema Integrado de Orçamentos foi desenvolvido para simplificar e otimizar o processo de precificação de grades metálicas, oferecendo ferramentas para gerenciar templates, custos de materiais e serviços, e configurações de impostos e lucros.

O sistema é dividido em várias seções acessíveis através das abas de navegação principais:

  • Templates: Para criar e gerenciar modelos de grades.
  • Preços: Onde você define os custos de matéria-prima e serviços.
  • Itens: Para adicionar grades específicas ao seu orçamento usando os templates criados.
  • Memória: Exibe os cálculos detalhados de custos por template.
  • Relatórios: Gera relatórios individuais e consolidados dos materiais e planos de corte.
  • Pricing: Permite configurar margens, despesas e impostos (incluindo IPI e ICMS) para chegar ao preço de venda.
  • Orçamento: Apresenta o orçamento comercial final com os preços calculados.

Atenção: O sistema utiliza o armazenamento local do seu navegador. Não há um banco de dados externo, o que significa que seus dados são salvos diretamente no seu dispositivo. Faça backups regulares se necessário (funcionalidade de exportação em desenvolvimento na aba de orçamentos).

2. Gerenciamento de Orçamentos Múltiplos

O sistema permite trabalhar com múltiplos orçamentos simultaneamente, cada um com seus próprios templates, itens e configurações de preços.

2.1. Criar um Novo Orçamento

Para iniciar um novo orçamento:

  1. Clique no botão Novo Orçamento localizado no cabeçalho.
  2. Um novo orçamento será criado com um nome padrão (ex: "Orçamento 2").
  3. O sistema automaticamente mudará para a aba do novo orçamento.

2.2. Alternar entre Orçamentos

Você pode alternar entre os orçamentos abertos no cabeçalho:

  1. Clique na aba do orçamento desejado (ex: Orçamento 1).
  2. O conteúdo da aplicação será atualizado para exibir os dados do orçamento selecionado.

2.3. Renomear um Orçamento

Para dar um nome mais descritivo ao seu orçamento:

  1. Com o orçamento ativo, clique no botão Renomear ao lado do nome atual.
  2. Uma janela modal será aberta.
  3. Digite o novo nome desejado no campo "Nome do Orçamento".
  4. Clique em Salvar para confirmar.

2.4. Fechar um Orçamento

Para remover um orçamento da sua lista:

  1. Clique no botão (X) na aba do orçamento que deseja fechar.
  2. Se o orçamento contiver dados, o sistema solicitará uma confirmação.
  3. Atenção: Não é possível fechar o último orçamento. Você deve ter pelo menos um orçamento ativo.

2.5. Duplicar Orçamento

Para criar uma cópia exata de um orçamento existente:

  1. Atualmente, a funcionalidade de duplicar orçamento não possui um botão direto na interface do usuário. No entanto, ela está implementada no código em multiBudgetManager.duplicateBudget().
  2. Em uma futura atualização, esta opção estará disponível na interface.

2.6. Exportar e Importar Orçamento

Para fazer backup ou transferir orçamentos entre dispositivos:

  1. Atualmente, as funcionalidades de exportar e importar orçamento não possuem botões diretos na interface do usuário. No entanto, elas estão implementadas no código em multiBudgetManager.exportBudget() e multiBudgetManager.importBudget().
  2. Em uma futura atualização, estas opções estarão disponíveis na interface, provavelmente na aba de "Relatórios" ou em um menu de gerenciamento de orçamentos.

Nota: A exportação de dados está formatada como JSON e inclui metadados, templates, itens, preços e configurações de pricing.

3. Templates de Grades Metálicas

A aba Templates é onde você define os modelos das grades que serão utilizadas nos orçamentos. Um template descreve as características físicas da grade.

3.1. Criar ou Editar um Template

Para criar um novo template:

  1. Navegue até a aba Templates.
  2. Preencha os campos no formulário "Criar/Editar Template".
  3. Clique em Salvar Template.

Para editar um template existente:

  1. Na seção "Templates Salvos", clique no botão (Editar) ao lado do template desejado.
  2. O formulário será preenchido com os dados do template selecionado.
  3. Faça as alterações necessárias e clique em Atualizar Template.

Campos do Template:

  • Nome do Template: Nome único para identificação.
  • Barras Portantes (Espessura e Largura): Dimensões das barras que suportam o peso.
  • Barras de Ligação (Diâmetro / Espessura Chata / Largura Chata):
    • Você deve preencher um tipo de barra de ligação. Se for redonda, informe o diâmetro. Se for chata, informe espessura e largura.
    • Não é possível usar barra redonda e chata simultaneamente.
  • Barras de Fechamento (Espessura e Largura): Dimensões das barras que fecham a grade.
  • Configuração da Malha (Malha Menor e Malha Maior): Distância entre as barras.
  • Parâmetros Gerais:
    • Densidade (kg/m³): Densidade do material (padrão: 7850 kg/m³ para aço).
    • Margem de Segurança (%): Margem aplicada ao peso final para compensar perdas. (padrão: 2%).
  • Usar preços customizados para este template: Marque esta opção para definir preços de Mão de Obra e Galvanização específicos para este template, sobrescrevendo os preços globais.

3.2. Configurando Templates como Degraus

Para configurar um template como degrau, marque a opção Configurar como Degrau.

Ao marcar esta opção, campos adicionais serão exibidos:

  • Chapa Lateral (Espessura e Largura):
    • A largura pode ser calculada automaticamente com base na "Largura Portante" e uma fórmula: ARRED.PARA.CIMA((Larg.Portante + 37) / 5) * 5.
  • Chapa Xadrez (Opcional):
    • Marque Incluir chapa xadrez para adicionar campos de espessura, largura e ajuste de peso.
    • A largura da chapa xadrez pode ser calculada automaticamente: Larg.Portante + Malha Menor.
    • O Ajuste de Peso é um multiplicador para o peso da chapa xadrez (padrão: 1.05).

Nota: Templates configurados como degraus não possuem Barras de Fechamento Laterais, que são substituídas pela Chapa Lateral.

3.3. Validações Importantes ao Criar/Editar Templates

O sistema possui validações para garantir a consistência dos dados do template:

  • Nome do template é obrigatório e deve ter entre 1 e 50 caracteres.
  • Espessura e Largura das Barras Portantes e de Fechamento devem ser maiores que 0.
  • Malha Menor deve ser menor que Malha Maior.
  • Espessura da barra portante não pode exceder 50% da malha menor.
  • Espessura do fechamento deve ser maior ou igual à espessura da barra portante.
  • Largura do fechamento deve ser maior ou igual à largura da barra portante.
  • Espessura do fechamento não pode exceder 50% da malha menor.
  • Diâmetro ou Espessura/Largura da Barra de Ligação devem ser definidos.
  • Diâmetro da ligação não pode exceder 50% da malha maior.
  • Espessura da ligação não pode exceder 50% da malha maior.
  • Se usar barra redonda, espessura e largura chata devem ser 0.
  • Se usar barra chata, diâmetro deve ser 0.
  • Para barra chata, ambas as dimensões (espessura e largura) devem ser maiores ou iguais a 2mm.
  • Se usar preços customizados, o campo "Mão de Obra (R$/kg)" é obrigatório.
  • Para templates de degrau, a espessura e largura da chapa lateral são obrigatórias.
  • Se incluir chapa xadrez, a espessura, largura e ajuste de peso da chapa xadrez são obrigatórios, e o ajuste de peso deve ser >= 1.

4. Configuração de Preços

A aba Preços permite definir os custos de matéria-prima e serviços de forma global ou específica por dimensão de material.

4.1. Configuração Global de Preços

Estes preços são aplicados a todos os cálculos, a menos que um template possua preços customizados.

Preencha os campos e clique em Salvar Preços.

Campos:

  • Matéria Prima (R$/kg):
    • Barra Chata Padrão
    • Barra Redonda Padrão
    • Chapa Lateral Padrão
    • Chapa Xadrez Padrão
  • Serviços (R$/kg):
    • Corte Chapa Lateral (Serviço de corte para chapas laterais)
    • Corte+Dobra Chapa Xadrez (Serviço de corte e dobra para chapas xadrez)
  • Processos (R$/kg):
    • Mão de Obra (Aplicado no peso final das grades (com margem de segurança))
    • Galvanização (Opcional - Aplicado no peso final das grades)

Você pode Resetar todos os preços globais para 0.

4.2. Preços Específicos por Material

É possível definir preços por kg para dimensões específicas de barras chatas, redondas e chapas, que terão precedência sobre os preços padrão globais.

Para adicionar um preço específico:

  1. Clique em um dos botões: Chata, Redonda, Chapa Lateral, ou Chapa Xadrez.
  2. Uma janela modal será aberta.
  3. Informe as dimensões (Espessura/Largura para chata/chapa, Diâmetro para redonda) e o Preço (R$/kg).
  4. Clique em Salvar.

Para editar um preço específico, clique no botão (Editar) ao lado do item na lista.

Para remover um preço específico, clique no botão (Remover) ao lado do item.

4.3. Como os Preços Afetam os Custos

Os custos de material são calculados com base no peso dos materiais e nos preços definidos.

  • Custo de Matéria Prima: É o peso bruto dos materiais (barras, chapas) multiplicado pelo preço por kg.
  • Custo de Mão de Obra: É o peso final das grades (já com a margem de segurança do template) multiplicado pelo preço de Mão de Obra por kg.
  • Custo de Galvanização: Similar à Mão de Obra, aplicado ao peso final com margem, se configurado.
  • Serviços de Chapas: Os serviços de corte e dobra para chapas laterais e xadrez são adicionados ao custo do material das chapas quando estas são utilizadas em templates de degrau.
  • Precedência de Preços: Se um preço específico for definido para uma dimensão de material, ele será usado no lugar do preço padrão global para aquele material. Se um template tiver "preços customizados", os valores de Mão de Obra e Galvanização desse template sobrescreverão os globais.

5. Itens do Orçamento

A aba Itens permite adicionar grades específicas ao seu orçamento usando os templates configurados.

5.1. Adicionar ou Editar um Item

Para adicionar um novo item ao orçamento:

  1. Navegue até a aba Itens.
  2. No formulário "Adicionar Item ao Orçamento":
    • Template: Selecione um dos templates criados previamente. Templates de degrau que já possuem um item no orçamento são desabilitados, pois aceitam apenas um item.
    • Quantidade: Número de grades a serem orçadas.
    • Largura (mm): Largura da grade em milímetros.
    • Comprimento (mm): Comprimento da grade em milímetros.
    • Descrição (Opcional): Uma descrição para o item (padrão: "Qtd grades LarguraxComprimento mm").
  3. Clique em Adicionar Item.

Para editar um item existente:

  1. Na seção "Itens do Orçamento", clique no botão (Editar) ao lado do item.
  2. O formulário será preenchido com os dados do item.
  3. Faça as alterações e clique em Atualizar Item.
  4. Você pode clicar em Cancelar Edição para descartar as alterações e retornar.

Para remover um item, clique no botão (Remover) ao lado dele.

Para limpar todos os itens do orçamento atual, clique em Limpar Orçamento.

5.2. Cálculos Básicos de um Item de Orçamento

Ao adicionar um item, o sistema realiza os seguintes cálculos, exibidos na lista de itens:

  • Área: Largura (m) × Comprimento (m) × Quantidade.
  • Peso (kg): Soma do peso de todas as barras e chapas necessárias para o item, antes da margem de segurança.
  • Peso c/ Margem (kg): Peso total multiplicado por (1 + Margem de Segurança do template / 100).
  • Custo Total: Soma dos custos de material, mão de obra, galvanização e serviços de chapas para o item.
  • Custo/unidade: Custo Total dividido pela Quantidade do item.

Validações de Item: O sistema verifica se o template selecionado existe, se as dimensões (largura e comprimento) são no mínimo 100mm, se a largura é pelo menos 2x a malha maior do template e o comprimento 2x a malha menor. Para degraus, o comprimento mínimo é 200mm.

5.3. Resumo do Orçamento

No lado direito da aba Itens, há um painel de resumo que consolida os totais do orçamento atual:

  • Total de Grades: Soma das quantidades de todos os itens.
  • Área Total: Soma das áreas de todas as grades em m².
  • Peso c/ Margem: Soma dos pesos com margem de todos os itens.
  • Custo Material: Custo total da matéria-prima (barras e chapas, incluindo serviços de corte/dobra de chapas).
  • TOTAL GERAL: Custo total de todos os itens, incluindo material, mão de obra e galvanização.
  • Consolidado de Materiais: Detalha o peso total por tipo de material (Ex: Barras Portantes, Barras Ligação, Chapa Lateral).

6. Memória de Cálculo Detalhada

A aba Memória apresenta uma análise aprofundada de como os custos são calculados para cada template utilizado no orçamento.

Para cada template com itens adicionados, a memória de cálculo inclui:

  • Visão Geral do Template: Nome, tipo (degrau, customizado), total de itens, área e peso com margem.
  • Itens do Template: Lista cada item que usa o template, com suas dimensões, peso e custo individual.
  • Barras Metálicas:
    • Lista detalhada das dimensões das barras (Espessura/Diâmetro, Largura).
    • Número de barras necessárias (de 6 metros, padrão).
    • Peso da matéria-prima (considerando a barra padrão de 6m).
    • Preço por kg.
    • Custo total de cada tipo de barra.
  • Chapas para Degraus (se aplicável):
    • Detalhes da Chapa Lateral e Chapa Xadrez (se incluída).
    • Dimensões, quantidade, peso e custo do material.
    • Nota: Esta seção mostra apenas o custo do material da chapa; os serviços de corte/dobra são detalhados na seção de Processos.
  • Processos:
    • Mão de Obra: Cálculo: Peso com Margem (kg) × Preço Mão de Obra (R$/kg).
    • Galvanização: Cálculo: Peso com Margem (kg) × Preço Galvanização (R$/kg).
    • Serviços de Chapas: Soma dos serviços de corte/dobra para chapas laterais e xadrez.
    • Total de Processos.
  • Total Geral: Custo total do template (material + processos).
  • Custo por m²: Custo total do template dividido pela área total dos itens que o utilizam.

7. Relatórios

A aba Relatórios fornece diferentes visões dos dados do seu orçamento.

7.1. Relatório Detalhado por Item

Este relatório lista cada item do orçamento com seus respectivos detalhes e custos.

  • Header: Descrição do item, dimensões, template utilizado, peso e custo total. Clique no cabeçalho para expandir/recolher os detalhes.
  • Detalhes do Material: Tabela com tipo de material (ex: Portante, Ligação, Fechamento), espessura/diâmetro, largura, comprimento, quantidade e peso.
  • Custos Detalhados: Material, Mão de Obra, Galvanização, Custo TOTAL, Custo/unidade e Custo/m².

7.2. Consolidado e Plano de Corte

Este relatório consolida os materiais necessários e gera um plano de corte otimizado para as barras.

Consolidado de Barras Metálicas:

  • Exibe o total de peças e peso total por template, espessura/diâmetro, largura e comprimento.

Consolidado de Chapas para Degraus (se aplicável):

  • Exibe o total de peças e peso total por template, tipo de chapa, espessura, largura e comprimento.

Plano de Corte:

Detalha como otimizar o corte das barras padrão (6000mm) para minimizar o desperdício para cada tipo de barra.

  • Para cada dimensão de barra, mostra o número de barras padrão necessárias.
  • Indica o percentual de desperdício (baixo, médio, alto).
  • Lista as peças a serem cortadas de cada barra.
  • Cálculo do Desperdício: É a soma das sobras de todas as barras dividida pelo comprimento total das barras utilizadas, expressa em percentual.

8. Configuração de Pricing

A aba Pricing é crucial para transformar o custo de produção em preço de venda, aplicando margens, despesas e impostos.

8.1. Tabela de Alíquotas

Esta seção permite visualizar e editar as alíquotas de ICMS (Alíquota Interna, Alíquota SP) e MVA (Margem de Valor Agregado para Grade, Grampo, Telas) por estado (UF).

Para editar, clique em Editar Alíquotas.

  • UF: Unidade Federativa.
  • MVA Grade / Grampo / Telas: Margem de Valor Agregado para Substituição Tributária.
  • Aliq. Int.: Alíquota Interna de ICMS do estado.
  • Aliq. SP: Alíquota de ICMS para vendas para São Paulo.

Após editar, clique em Salvar Alterações.

Você pode restaurar as alíquotas de uma UF específica clicando no botão (Restaurar) na linha da UF, ou restaurar todas as alíquotas para o padrão clicando em Restaurar Padrão.

Atenção: Alterações nas alíquotas afetam todos os cálculos de pricing. Use dados oficiais e atualizados.

8.2. Configuração de Pricing por Template

Cada template no seu orçamento pode ter sua própria configuração de pricing.

O painel superior (Toolbar de Pricing) mostra um resumo dos templates: total, configurados, calculados e com itens.

Você pode copiar a configuração de um template já configurado para outros que ainda não possuem configuração, usando os campos "Copiar de..." e o botão Copiar para Não Configurados.

O botão Calcular Todos recalcula o pricing para todos os templates que têm itens no orçamento e estão configurados.

Painel de Configuração Individual por Template:

Cada "Template Pricing Card" representa um template e pode ser expandido/recolhido para configurar seus parâmetros de pricing.

  • Status do Template:
    • Sem Itens: O template não foi usado em nenhum item do orçamento.
    • Não Configurado: O template possui itens, mas sua configuração de pricing ainda não foi salva.
    • Configurado: A configuração foi salva, mas o preço final ainda não foi calculado (ou está desatualizado).
    • Calculado: O preço final foi calculado com sucesso.
  • Tipo de Configuração: Essencial para determinar como o ICMS será aplicado (Venda de Grade ST, Venda para Não Contribuinte, etc.).
  • UF de Destino: Estado para onde o produto será vendido, usado para buscar as alíquotas corretas.
  • Percentuais de Negócio:
    • Lucro Líquido (%), Despesas Fixas (%), Comissões Internas (%), Comissões Representantes (%), Despesas Financeiras (%), CSLL/IRPJ (%).
  • ICMS Material (%): Alíquota de ICMS sobre a compra da matéria-prima.
  • Configuração de IPI: Esta é uma seção crítica com regras específicas.
  • Configuração de Frete: Frete Percentual (%) e Frete Valor Fixo (R$).
  • Substituição Tributária (ST Compra): Quantidade e Valor Unitário (R$) para cálculo de ST na compra.
  • Taxa de Energia: Quantidade e Valor Unitário (R$) para cálculo de taxa de energia.

Após ajustar os parâmetros, clique em Calcular Preço dentro do card do template para obter o resultado.

O resultado do cálculo mostrará: Preço de Venda, Preço por m², Valor IPI e Valor com IPI+ST.

8.3. Regras de IPI (Imposto sobre Produtos Industrializados)

A configuração do IPI é complexa e o sistema implementa validações rigorosas:

  • Se IPI Material > 0%:
    • IPI na Revenda deve ser 0%.
    • IPI Venda Final deve ser > 0%.
    • Esta configuração geralmente indica uma operação de INDUSTRIALIZAÇÃO.
  • Se IPI na Revenda > 0%:
    • IPI Material deve ser 0%.
    • IPI Venda Final deve ser 0%.
    • Esta configuração geralmente indica uma operação de REVENDA.
  • Todos os campos de IPI (Material, Revenda, Venda Final) podem ser 0% simultaneamente.

O sistema exibirá mensagens de erro ou avisos se as regras não forem seguidas, destacando os campos com problemas.

8.4. Entendendo o Cálculo de Pricing (Motor de Cálculo)

O motor de cálculo de pricing (CalculationEngine.js) realiza uma série de etapas para determinar o preço de venda, utilizando as configurações do template e as alíquotas da UF de destino.

Principais Variáveis e Etapas do Cálculo:

As variáveis são nomeadas com B_ para facilitar a referência a uma possível planilha de cálculo.

  • B1_custoMaterial: Soma dos custos de material puro do template (sem serviços de chapas).
  • B3_ipiMaterial: IPI sobre a compra de material.
  • B2_custoMaterialComIPI: B1_custoMaterial * (1 + B3_ipiMaterial).
  • B4_valorIPIMaterial: Valor do IPI sobre material (pode ser 0 para venda para estaleiro ou exportação).
  • B5_icmsMaterial: Alíquota de ICMS sobre material.
  • B6_valorICMSMaterial: Valor do ICMS sobre material (pode ser 0 para venda para estaleiro ou exportação).
  • B7_ipiRevenda: IPI na revenda.
  • B8_valorIPIRevenda: Valor do IPI na revenda.
  • B11_somaServicos: Soma da Mão de Obra, Galvanização e Serviços de Chapas.
  • B12_servicoTotal: Considera B11_somaServicos.
  • B28_stCompraTotal: Custo total da ST na compra.
  • B32_taxaEnergiaTotal: Custo total da taxa de energia.
  • B34_custoLiquido: Custo total líquido, incluindo material com IPI, serviços, ST compra e taxa de energia, menos IPI e ICMS de material (se aplicável).

Cálculo do Preço de Venda Base (B94_valorMercadoria):

  • Este é um cálculo crucial, que "volta" o preço a partir do custo líquido e de todos os percentuais sobre a venda.
  • Ele considera o Lucro Líquido, Despesas Fixas, Comissões, Despesas Financeiras, CSLL/IRPJ, ICMS de Operação Própria, COFINS, PIS e Frete Percentual.
  • B92_percAntesMercadoria = Soma de todos os percentuais sobre a venda (exceto frete fixo).
  • B94_valorMercadoria = (B34_custoLiquido / (1 - B92_percAntesMercadoria)) + B91_freteValor (Frete Valor Fixo).

Cálculo dos Impostos sobre a Venda:

  • B99_valorIPI: IPI sobre a venda final (se B98_ipiVenda > 0). Não aplicado para venda para estaleiro ou exportação.
  • B35_baseICMSST: Soma das MVAs de Grade, Grampo e Telas, se aplicável.
  • B36_baseICMSSTTotal: Base de cálculo do ICMS ST ((B94_valorMercadoria + B99_valorIPI) * B35_baseICMSST).
  • B38_icmsSTBruto: Soma das alíquotas internas para ST (Grade, Grampo, Telas).
  • B39_icmsSTBrutoTotal: ICMS ST Bruto (B36_baseICMSSTTotal * B38_icmsSTBruto).
  • B82_icmsOpPropria: Alíquota de ICMS de Operação Própria (Soma de Aliq. SP ou Aliq. Int. para não contribuinte/consumidor final/revenda normal).
  • B83_valorICMSOpPropria: Valor do ICMS de Operação Própria.
  • B84_cofins e B86_pis: Alíquotas de COFINS e PIS. Não aplicadas para venda para estaleiro ou exportação.
  • B85_valorCOFINS e B87_valorPIS: Valores de COFINS e PIS (calculados sobre B94_valorMercadoria - B83_valorICMSOpPropria).
  • B95_difal: Diferencial de Alíquota (DIFAL) para não contribuinte.
  • B96_valorDifal: Valor do DIFAL.
  • B97_precoVendaComDifal: B94_valorMercadoria + B96_valorDifal. Este é o Preço de Venda exibido.
  • B101_valorDifST: Diferencial de ST (B39_icmsSTBrutoTotal - B83_valorICMSOpPropria).
  • B102_valorFinalComIPIST: B97_precoVendaComDifal + B99_valorIPI + B101_valorDifST. Este é o Valor com IPI + ST exibido.

Nota: O cálculo de pricing é complexo e simula um cenário tributário para grades metálicas. Consulte um especialista fiscal para validação dos dados em um cenário real. O sistema não trata cenários específicos como vendas para estaleiros ou exportação, onde algumas isenções podem ser aplicadas (B68 e B69).

9. Orçamento Comercial Final

A aba Orçamento apresenta um resumo do orçamento comercial, utilizando os preços calculados na seção de Pricing.

Para que os valores sejam exibidos, você deve ter itens adicionados ao orçamento e ter executado o cálculo de pricing para os templates utilizados (botão Calcular Preço na aba Pricing ou Calcular Todos).

Conteúdo do Relatório:

  • Resumo Superior: Total com DIFAL e Total com IPI+ST.
  • Tabela de Itens:
    • Item (Descrição e Template)
    • Quantidade
    • Dimensões
    • Área (m²)
    • Valor Unitário (calculado)
    • Valor Total (calculado)
  • Totais Finais: Total com DIFAL e Total com IPI+ST novamente no rodapé da tabela.

10. Dicas e Validações para Uso Correto

  • Campos Obrigatórios: Campos marcados com um asterisco (*) são obrigatórios.
  • Validação em Tempo Real: Muitos campos possuem validação em tempo real. Erros são indicados por uma borda vermelha e uma mensagem de ajuda em vermelho.
  • Mensagens de Notificação: O sistema exibirá mensagens no canto superior direito da tela para informar sobre o sucesso de operações, avisos ou erros.
  • Limpeza de Formulários: Utilize os botões "Limpar" para resetar os campos dos formulários.
  • Formato de Números: Insira valores numéricos. O sistema aceita vírgulas ou pontos como separador decimal e fará a conversão interna.
  • Backup de Dados: Como os dados são salvos localmente, considere exportar seus orçamentos periodicamente como backup.
  • Revisão de Validações: Sempre preste atenção às mensagens de erro e avisos, especialmente nas seções de Templates e Pricing, pois elas indicam problemas que podem afetar a precisão dos cálculos.
  • Unicidade de Degrau: Lembre-se que templates de degrau só aceitam um item no orçamento.
  • Ordem das Operações: A sequência lógica de uso é:
    1. Configurar Templates.
    2. Configurar Preços Globais e/ou Específicos.
    3. Adicionar Itens ao Orçamento.
    4. Visualizar Memória de Cálculo e Relatórios (opcional, para conferência).
    5. Configurar Pricing por Template e Calcular os Preços.
    6. Gerar Orçamento Comercial Final.

11. Fluxo de Uso Completo - Exemplo Prático

Vamos percorrer um cenário completo de uso do sistema:

Etapa 1: Preparar o Orçamento e Configurar Templates

  1. Crie um Novo Orçamento (se necessário): No cabeçalho, clique em para criar um novo orçamento. Você pode renomeá-lo para "Orçamento Cliente ABC".
  2. Crie um Template de Grade Padrão:
    • Vá para a aba Templates.
    • Preencha o formulário:
      • Nome do Template: Grade Padrão
      • Barras Portantes: Esp. 4.76mm, Larg. 32mm
      • Barras de Ligação: Diâmetro 6.35mm (deixe Esp. Chata e Larg. Chata como 0)
      • Barras de Fechamento: Esp. 4.76mm, Larg. 32mm
      • Malha Menor: 33mm, Malha Maior: 100mm
      • Densidade: 7850, Margem: 2
    • Clique em Salvar Template.
  3. Crie um Template de Degrau:
    • Ainda na aba Templates, limpe o formulário (botão Limpar).
    • Preencha:
      • Nome do Template: Degrau Padrão
      • Barras Portantes: Esp. 4.76mm, Larg. 38mm
      • Barras de Ligação: Diâmetro 6.35mm
      • Barras de Fechamento: Esp. 4.76mm, Larg. 38mm
      • Malha Menor: 33mm, Malha Maior: 100mm
      • Marque Configurar como Degrau.
      • Chapa Lateral: Esp. 4.76mm, Larg. 80mm (largura calculada automaticamente).
      • Marque Incluir chapa xadrez.
      • Chapa Xadrez: Esp. 3.18mm, Larg. 71mm (largura calculada), Ajuste Peso: 1.05.
    • Clique em Salvar Template.

Etapa 2: Configurar Preços

  1. Vá para a aba Preços.
  2. Preços Globais:
    • Barra Chata Padrão: 12.00 R$/kg
    • Barra Redonda Padrão: 13.00 R$/kg
    • Chapa Lateral Padrão: 14.00 R$/kg
    • Chapa Xadrez Padrão: 16.00 R$/kg
    • Corte Chapa Lateral: 2.00 R$/kg
    • Corte+Dobra Chapa Xadrez: 3.00 R$/kg
    • Mão de Obra: 15.00 R$/kg
    • Galvanização: 8.00 R$/kg
    • Clique em Salvar Preços.
  3. Preço Específico (Exemplo - Opcional):
    • Clique em Chata.
    • Espessura: 4.76, Largura: 32, Preço: 12.50 R$/kg.
    • Clique em Salvar. Este preço será usado para barras chatas 4.76x32mm em vez dos 12.00 R$/kg padrão.

Etapa 3: Adicionar Itens ao Orçamento

  1. Vá para a aba Itens.
  2. Adicione um Item "Grade Padrão":
    • Template: Grade Padrão
    • Quantidade: 10
    • Largura: 900mm, Comprimento: 1200mm
    • Descrição: Grades para plataforma
    • Clique em Adicionar Item.
  3. Adicione um Item "Degrau Padrão":
    • Template: Degrau Padrão
    • Quantidade: 20
    • Largura: 300mm, Comprimento: 800mm
    • Descrição: Degraus para escada
    • Clique em Adicionar Item.

Etapa 4: Análise de Custos e Relatórios

  1. Memória de Cálculo:
    • Vá para a aba Memória.
    • Analise como os custos de material, mão de obra, galvanização e serviços de chapa são detalhados para cada template (Grade Padrão e Degrau Padrão).
  2. Relatórios:
    • Vá para a aba Relatórios.
    • Examine o "Relatório Detalhado por Item" para ver os materiais e custos de cada grade individualmente.
    • Verifique o "Consolidado e Plano de Corte" para o resumo dos materiais e a otimização de corte das barras.

Etapa 5: Configurar Pricing e Gerar Orçamento Final

  1. Configurar Alíquotas (se necessário):
    • Vá para a aba Pricing.
    • Se sua UF não estiver correta, clique em Editar Alíquotas, ajuste e salve.
  2. Configurar Pricing para o Template "Grade Padrão":
    • No card do "Grade Padrão", expanda-o (clique no cabeçalho).
    • Tipo de Configuração: Venda de Grade ST
    • UF de Destino: SP (ou sua UF)
    • Lucro Líquido: 20%, Despesas Fixas: 9.5%, Comissões Internas: 3.5%, CSLL/IRPJ: 2.81% (use os valores padrão se preferir).
    • ICMS Material: 18%, IPI Material: 3.25%, IPI Venda Final: 3.25% (deixe IPI Revenda como 0).
    • Clique em Calcular Preço para este template.
  3. Configurar Pricing para o Template "Degrau Padrão":
    • No card do "Degrau Padrão", expanda-o.
    • Tipo de Configuração: Venda de Grade ST (ou o tipo adequado para degraus)
    • UF de Destino: SP (ou sua UF)
    • Ajuste os percentuais conforme necessário.
    • Configure o IPI seguindo as regras (ex: IPI Material 3.25%, IPI Venda Final 3.25%).
    • Clique em Calcular Preço para este template.
  4. Gerar Orçamento Comercial Final:
    • Vá para a aba Orçamento.
    • Você verá a tabela com os itens do seu orçamento, agora com os valores unitários e totais calculados com base no pricing.
    • Os totais com DIFAL e com IPI+ST serão exibidos no topo e rodapé.

Parabéns! Você concluiu um fluxo completo de orçamento. Agora você pode explorar as funcionalidades para suas necessidades específicas.
